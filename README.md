# 🔐 AI-Powered Intrusion Detection & Response System

## 🚀 Overview

This project is an autonomous cybersecurity system that detects suspicious login activity, analyzes threats using AI, and automatically responds by scanning and blocking malicious IPs.

## 🧠 Features

- Log parsing & analysis
- Brute force detection
- AI-based threat analysis (OpenRouter - LLaMA 3)
- Multi-agent architecture:
  - Detection Agent
  - Response Agent
  - Action Agent
- Automated response:
  - Port scanning
  - IP blocking
- Web dashboard (Flask)

## 🏗️ Architecture

Logs → Detection → Response → Action → AI Analysis

## 🛠️ Tech Stack

- Python
- Flask
- AI API (OpenRouter)
- HTML/CSS

## 📸 Output

- Detects brute force attacks
- Identifies attacker IP
- Automatically blocks attacker
- Provides AI explanation

## ⚠️ Note

This project is for educational purposes and simulates cybersecurity defense mechanisms.
